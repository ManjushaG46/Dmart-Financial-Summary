# D-Mart Financial & Sales Performance Dashboard (Power BI)

Project Purpose
The purpose of this project is to analyze D-Mart’s financial and sales performance across different regions, product categories, customer types, and time periods.
This dashboard helps stakeholders to:
Track overall revenue, profit, cost, and budget performance
Identify high-performing regions and product categories
Monitor monthly revenue trends and budget variance
Understand customer-type contribution to total profit
The dashboard supports data-driven decision-making by providing clear, interactive, and visual insights.

Tools & Technologies Used
Power BI – Data modeling, DAX measures, and interactive dashboards
DAX (Data Analysis Expressions) – KPIs, profit margin, variance calculations
Microsoft Excel / CSV – Initial data handling
GitHub – Project documentation and version control

Data Source
Kaggle

This Power BI dashboard provides a complete financial and sales overview of D-Mart operations. It enables quick performance tracking, trend identification, and strategic insights through an intuitive and interactive interface.
